# News Crawler

新闻爬虫项目 - 自动抓取和分析新闻数据

## 环境

- Python 3.10
- Conda 环境：`news_crawler`

## 安装

```bash
conda activate news_crawler
pip install -r requirements.txt
```

## 使用

```bash
python src/main.py
```

## 项目结构

```
news_crawler/
├── src/
│   ├── main.py
│   ├── crawler.py
│   └── parser.py
├── data/
├── tests/
├── requirements.txt
└── README.md
```
